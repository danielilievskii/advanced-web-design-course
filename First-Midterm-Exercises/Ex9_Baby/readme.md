За потребите на оваа задача потребно е да го ископирате решението од претходната задача (класата Human).

Напишете Baby конструктор што наследува од Human. Покрај името и возраста, бебето чува вредност за својата омилена играчка. Покрај методите на Human.prototype, бебињата имаат способност да си играат (play()). Тука треба да се испечати "Baby Joe, 1 is playing with Teddy Bear".

**For example:**

**Input**

const baby = new Baby("Alice", 1, "Bear");

console.log(baby.play());

**Result**

Baby Alice, 1 is playing with Bear
