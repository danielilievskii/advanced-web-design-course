Да се напише функција merge која ќе спојува 2 полиња во JavaScript. Функцијата треба да го врати резултантното поле.

Спојувањето треба да се изврши како на примерот:

**For example:**

**Input**

1, 2, 3, 4, 5

100, 200, 300, 400, 500

**Result**

[ [ '1', '100' ],

[ '2', '200' ],

[ '3', '300' ],

[ '4', '400' ],

[ '5', '500' ] ]

**For example 2:**

**Input**

1, 2, 3, 4, 5

100, 200, 300

**Result**

[ [ '1', '100' ],

[ '2', '200' ],

[ '3', '300' ],

[ '4', undefined ],

[ '5', undefined ] ]
